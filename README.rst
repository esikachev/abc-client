abc-client
==========

This is the client for abc-server located at https://github.com/esikachev/abc-server.

**How to use:**

1. Install abc-server. Instruction is here: https://github.com/esikachev/abc-server/blob/master/README.rst

2. Run command 

.. sourcecode:: console

    $ tox -e venv -- abc --init
..
